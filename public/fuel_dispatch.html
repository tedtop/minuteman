Troubleshooting

If dispatch data stops loading: call QT Technologies Client Support at 866-788-3241 or email QTHelpDesk@QTTechnologies.com, identify as Ted, and ask for a password reset and/or unlock on the TTMSO account. This re-enables the existing Company Location ID / User ID UUIDs without needing to change them.

If the Company Location ID or User ID ever need to change: these are Vercel environment variables (Project Settings → Environment Variables, or the vercel env CLI) and require a redeploy. This cannot be done from within this app.
